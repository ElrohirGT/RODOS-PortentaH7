# RODOS-PortentaH7

[Portenta H7](https://www.arduino.cc/pro/hardware/product/portenta-h7) is a high
performance board released by [Arduino](https://www.arduino.cc).

## How to Compile?

Execute the following commands being inside the "rodos" directory:

```sh
source setenvs.sh
rodos-lib.sh cm4
```
